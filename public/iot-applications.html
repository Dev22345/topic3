IoT Applications

1) Iot Apps

A) What is an Iot application?

-> An IoT application or Internet of Things are applications which are used to control the hardware through the use of a simple interface build using web technologies such as python, html, css, js etc.

  • Through this application the user is able to analyse and monitor sensor data.
  • These applications can be accessed, used and control the hardware by a multiple number of users.

    • B) What are the main features needed?

  • Data Security
  • Realtime Data monitoring
  • Scalability
  • Integrtion
  • Optimization
  • Data Backup
  • User Interface
  • Error reports notification

    • C) What are some examples?

  • Smart home security system
  • Smart evacuation system
  • Health Monitoring system
  • Hospital or Resturant Reservation system
  • Appliance monitor and controller

    • 2) Smart Building

    A) What does smart mean?

    ->Smart building refers to a building which is automated to a certain extent in order to reduce human effort and control appliances using an interface while also covering faults that may occur during the whole process.
    It also covers resource conservation, security and health of the users.

    B) What are the main features needed?

  • Data Handaling
  • Sensor data precision and accuracy
  • Realtime health monitoring
  • Fault notifying system
  • Health preserving
  • Connectivity
  • Abstraction
  • Automation

    • C) What are some smart-Building scenarios?

  • Security
  • Congestion Avoidance detection
  • Air-Quality monitoring and ventilation
  • Turing on or off appliances automatically based on its usage: Saving Electricity
  • Contactless Appliances
  • Smart Evacuation System
